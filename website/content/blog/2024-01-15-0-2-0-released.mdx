# Super.js 0.2.0 Released

Version 0.2.0 is the biggest release yet.

## Gradual Typing

You can now mix typed and untyped code freely:

```typescript
function greet(name: string): string {
  return `Hello, ${name}!`
}

function add(a, b) {
  return a + b
}
```

## VS Code Extension

Install the Super.js VS Code extension for syntax highlighting, 13 snippets, and bracket matching.

## What's Next

Version 0.3.0 will focus on the standard library. See the [roadmap](/docs/roadmap) for details.
