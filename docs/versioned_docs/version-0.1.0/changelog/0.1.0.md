# Super.js 0.1.0

**Release Date:** January 1, 2024

This is the first stable release of Super.js, bringing a complete, production-ready superset of JavaScript with built-in type safety, formatting, linting, and testing capabilities.

## 🎉 What's New

### Core Language Features

- **Type Annotations and Type Inference**
  - Full support for static type annotations
  - Smart type inference based on ECMA standards
  - Type checking for all JavaScript constructs

- **Interface and Class Definitions**
  - TypeScript-compatible interface syntax
  - Class inheritance with type safety
  - Method and property type checking

- **Generic Types and Constraints**
  - Generic functions and classes
  - Type constraints and bounds
  - Generic type inference

- **Union and Intersection Types**
  - Union types for flexible type definitions
  - Intersection types for type composition
  - Type narrowing with type guards

- **Type Aliases and Utility Types**
  - Custom type aliases for better code organization
  - Built-in utility types (`Partial`, `Required`, `Pick`, etc.)
  - Conditional and mapped types

### Built-in Tooling

- **Super.js Compiler (`superjs build`)**
  - Fast compilation from `.sjs` to `.js`

- **Linter (`superjs lint`)**
  - Static analysis and error detection
  - Type-aware linting rules
  - Performance and security checks

- **Testing Framework (`superjs test`)**
  - Built-in testing framework
  - Jest-compatible API
  - Test coverage reporting

- **Development Server (`superjs dev`)**
  - Hot reloading for development
  - Fast compilation for quick feedback

  - Error overlay and debugging support

## 🚀 Quick Start

### Create Your First Super.js File

```javascript
// hello.sjs
interface Greeting {
  message: string;
  recipient: string;
}

function createGreeting(recipient: string): Greeting {
  return {
    message: "Hello",
    recipient: recipient
  };
}

function formatGreeting(greeting: Greeting): string {
  return `${greeting.message}, ${greeting.recipient}!`;
}

// Usage
const greeting = createGreeting("World");
const formattedMessage = formatGreeting(greeting);
console.log(formattedMessage);
```

### Compile and Run

```bash
# Compile the file
superjs build --source hello.sjs

# Run the compiled JavaScript
node hello.js
```

## 🛠️ Development Workflow

### Development Mode

```bash
# Start development server with hot reloading
superjs dev

# Watch for changes and recompile
superjs build --source src/ --watch
```

### Testing

```bash
# Run all tests
superjs test

# Run tests with coverage
superjs test --coverage

# Run tests in watch mode
superjs test --watch
```

### Code Quality

```bash
# Format code
superjs format

# Lint code
superjs lint

# Lint with auto-fix
superjs lint --fix
```
