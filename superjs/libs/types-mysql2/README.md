# @superjs/types-mysql2

Typed SuperJS bindings for [`mysql2`](https://www.npmjs.com/package/mysql2) 3.x.

## Install

```bash
pnpm add mysql2 @superjs/types-mysql2
```

## Usage

```sjs
import type { Pool } from "@superjs/types-mysql2"
```
