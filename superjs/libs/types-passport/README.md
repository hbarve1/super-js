# @superjs/types-passport

Typed SuperJS bindings for [`passport`](https://www.npmjs.com/package/passport) 0.7.x.

## Install

```bash
pnpm add passport @superjs/types-passport
```

## Usage

```sjs
import type { PassportStatic } from "@superjs/types-passport"
```

See [STATUS.md](./STATUS.md) for coverage and known gaps.
