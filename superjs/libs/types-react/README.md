# @superjs/types-react

Typed SuperJS bindings for [`react`](https://www.npmjs.com/package/react) 18.x.

## Install

```bash
pnpm add react @superjs/types-react
```

## Usage

```sjs
import type { ReactElement } from "@superjs/types-react"
```
