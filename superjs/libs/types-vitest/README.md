# @superjs/types-vitest

Typed SuperJS bindings for [`vitest`](https://www.npmjs.com/package/vitest) 1.x.

## Install

```bash
pnpm add vitest @superjs/types-vitest
```

## Usage

```sjs
import type { TestContext } from "@superjs/types-vitest"
```
