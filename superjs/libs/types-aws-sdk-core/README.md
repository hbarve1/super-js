# @superjs/types-aws-sdk-core

Typed SuperJS bindings for `@aws-sdk/client-*` 3.x.

## Install

```bash
pnpm add <runtime-package> @superjs/types-aws-sdk-core
```

## Usage

```sjs
import type { SmithyClient } from "@superjs/types-aws-sdk-core"
```
