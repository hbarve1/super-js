# @superjs/types-nextjs

Typed SuperJS bindings for [`next`](https://www.npmjs.com/package/next) 14.x.

## Install

```bash
pnpm add next @superjs/types-nextjs
```

## Usage

```sjs
import type { NextRequest } from "@superjs/types-nextjs"
```
