# @superjs/types-redis

Typed SuperJS bindings for [`ioredis`](https://www.npmjs.com/package/ioredis) 5.x.

## Install

```bash
pnpm add ioredis @superjs/types-redis
```

## Usage

```sjs
import type { Redis } from "@superjs/types-redis"
```
