# @superjs/types-prisma

Typed SuperJS bindings for [`@prisma/client`](https://www.npmjs.com/package/@prisma/client) 5.x.

## Install

```bash
pnpm add @prisma/client @superjs/types-prisma
```

## Usage

```sjs
import type { PrismaClient } from "@superjs/types-prisma"
```
