# @superjs/types-winston

Typed SuperJS bindings for [`winston`](https://www.npmjs.com/package/winston) 3.x.

## Install

```bash
pnpm add winston @superjs/types-winston
```

## Usage

```sjs
import type { Logger } from "@superjs/types-winston"
```

See [STATUS.md](./STATUS.md) for coverage and known gaps.
