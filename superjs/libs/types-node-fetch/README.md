# @superjs/types-node-fetch

Typed SuperJS bindings for [`node-fetch`](https://www.npmjs.com/package/node-fetch) 3.x.

## Install

```bash
pnpm add node-fetch @superjs/types-node-fetch
```

## Usage

```sjs
import type { Response } from "@superjs/types-node-fetch"
```
