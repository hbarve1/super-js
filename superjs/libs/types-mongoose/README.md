# @superjs/types-mongoose

Typed SuperJS bindings for [`mongoose`](https://www.npmjs.com/package/mongoose) 8.x.

## Install

```bash
pnpm add mongoose @superjs/types-mongoose
```

## Usage

```sjs
import type { Model } from "@superjs/types-mongoose"
```
