# @superjs/types-jsonwebtoken

Typed SuperJS bindings for [`jsonwebtoken`](https://www.npmjs.com/package/jsonwebtoken) 9.x.

## Install

```bash
pnpm add jsonwebtoken @superjs/types-jsonwebtoken
```

## Usage

```sjs
import type { JwtPayload } from "@superjs/types-jsonwebtoken"
```
