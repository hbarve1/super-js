# @superjs/types-dotenv

Typed SuperJS bindings for [`dotenv`](https://www.npmjs.com/package/dotenv) 16.x.

## Install

```bash
pnpm add dotenv @superjs/types-dotenv
```

## Usage

```sjs
import type { DotenvConfigOutput } from "@superjs/types-dotenv"
```

See [STATUS.md](./STATUS.md) for coverage and known gaps.
