# @superjs/types-axios

Typed SuperJS bindings for [`axios`](https://www.npmjs.com/package/axios) 1.x.

## Install

```bash
pnpm add axios @superjs/types-axios
```

## Usage

```sjs
import type { AxiosInstance } from "@superjs/types-axios"
```
