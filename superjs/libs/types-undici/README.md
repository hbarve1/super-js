# @superjs/types-undici

Typed SuperJS bindings for [`undici`](https://www.npmjs.com/package/undici) 6.x.

## Install

```bash
pnpm add undici @superjs/types-undici
```

## Usage

```sjs
import type { Dispatcher } from "@superjs/types-undici"
```
