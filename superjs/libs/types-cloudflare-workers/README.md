# @superjs/types-cloudflare-workers

Typed SuperJS bindings for `@cloudflare/workers-types` 4.x.

## Install

```bash
pnpm add <runtime-package> @superjs/types-cloudflare-workers
```

## Usage

```sjs
import type { Fetcher } from "@superjs/types-cloudflare-workers"
```

See [STATUS.md](./STATUS.md) for coverage and known gaps.
