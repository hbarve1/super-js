# Super.js Language Support for VS Code

This extension adds language support for Super.js (`.sjs`) files in Visual Studio Code.

## Features

- Syntax highlighting for Super.js files
- Support for JSX syntax
- Type system highlighting
- Smart bracket matching and auto-closing
- Code folding
- Comment toggling

## Supported File Extensions

- `.sjs` - Super.js source files

## Syntax Highlighting

The extension provides syntax highlighting for:

- Keywords and control flow statements
- Types and interfaces
- Classes and functions
- JSX elements and attributes
- Template literals
- Comments
- String literals
- Numeric literals
- Built-in objects and types

## Installation

1. Open VS Code
2. Press `Ctrl+P` (Windows/Linux) or `Cmd+P` (macOS)
3. Type `ext install superjs-vscode`
4. Press Enter

## Development

To work on this extension:

1. Clone the repository
2. Run `npm install`
3. Open in VS Code
4. Press F5 to start debugging
